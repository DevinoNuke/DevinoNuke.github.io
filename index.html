Proyek Terbaru

               
                   
                                               
                           

Final Project: Iontophoresis Therapy Device Based on IoT

                           

Dirancang dan dikembangkan perangkat terapi iontophoresis otomatis berbasis IoT untuk pengobatan hiperhidrosis palmar. Mengintegrasikan sensor Galvanic Skin Response (GSR) dan logika fuzzy Mamdani untuk menyesuaikan tegangan dan durasi berdasarkan kondisi kulit. Melakukan eksperimen pada pasien hiperhidrosis nyata dan responden simulasi untuk memvalidasi efektivitas perangkat. Mengembangkan aplikasi seluler untuk memantau data terapi.

                            Lihat Proyek                        
                   
                   
                                               
                           

Studi Kasus: Sistem Otomatisasi Industri dengan PLC Siemens

                           

Menganalisis, merancang, dan mengimplementasikan sistem kontrol otomatisasi menggunakan PLC Siemens S7-1200 untuk simulasi lini produksi. Mengembangkan ladder logic dan HMI (Human-Machine Interface) untuk pemantauan dan kontrol proses. Berhasil meningkatkan efisiensi operasional dan mengurangi intervensi manual dalam proses simulasi.

                            Lihat Studi Kasus                        
                   
                   
                                               
                           

Pengembangan Sistem Monitoring Lingkungan IoT

                           

Membuat prototipe sistem pemantauan lingkungan real-time menggunakan sensor suhu, kelembaban, dan kualitas udara (MQ-135) terintegrasi dengan ESP32. Data dikirimkan ke platform cloud (misalnya, Thingspeak) untuk visualisasi dan analisis. Proyek ini berfokus pada efisiensi energi dan akurasi data.
